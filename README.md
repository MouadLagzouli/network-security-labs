# Network Security Labs

Ce dépôt regroupe mes travaux pratiques et projets réalisés en réseaux informatiques, systèmes et cybersécurité.

## Objectif

L’objectif de ce dépôt est de présenter des configurations réseau réalisées dans un cadre académique et pratique, principalement avec Cisco Packet Tracer.

## Projets disponibles

| Projet | Description |
|---|---|
| NAT-PAT | Configuration du NAT statique, NAT dynamique et PAT avec Cisco Packet Tracer |

## Technologies utilisées

- Cisco Packet Tracer
- Routage IP
- NAT / PAT
- Adressage IP
- Réseaux LAN / WAN
- Tests de connectivité

## Compétences développées

- Configuration de routeurs Cisco
- Traduction d’adresses IP
- Analyse de la connectivité réseau
- Compréhension du passage d’un réseau privé vers un réseau public
- Simulation et dépannage réseau
