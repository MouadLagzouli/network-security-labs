# TP Réseau : VLAN et Routage Inter-VLAN

## Objectif du TP

Ce travail pratique a pour objectif de mettre en place une segmentation réseau à l’aide des VLANs et de permettre la communication entre ces VLANs grâce au routage inter-VLAN.

## Technologies utilisées

- Cisco Packet Tracer
- Switch Cisco
- Routeur Cisco
- VLAN
- Routage inter-VLAN
- Adressage IP
- Tests de connectivité

## Description

Dans ce TP, le réseau est divisé en plusieurs VLANs afin de séparer les différents groupes ou services d’un réseau local.

Le routage inter-VLAN permet aux machines appartenant à des VLANs différents de communiquer entre elles à travers un routeur ou un équipement de couche 3.
